大家好，

`wbot` 是个人量化交易机器人项目（Go，all-in-one）。本仓库用 **GitHub 留言驱动协作**：功能、计划、缺陷、发布，都以 GitHub 上的明确留言为触发源；其余执行尽量交给 Agent 自动化完成。

### 怎么用

- 先在 Issue / Discussion / PR 评论里留下**可引用的留言**（最好带清晰目标与边界）
- 开 PR 时，在描述里填写 `Driven-By` 并粘贴留言链接（`Source:`）
- 文档在 `doc/`，Obsidian 友好、短小、可双向链接
